# react-native-vehicle
Android-Auto and CarPlay library support for ReactNative
## Installation

```sh
npm install react-native-vehicle
```

## Usage

```js
import { AndroidAutoView } from "react-native-vehicle";

// ...

<AndroidAutoView color="tomato" />
```
